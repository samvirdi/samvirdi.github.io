Deltahacks V
Mapped Ideas

My friend and I decided to make this project for our 2019 Deltahacks V challenge. We dove into a type of development that we barely had any experience in which made this project even more fun. We made this web application in 20 hours by learning node.js via forums, StackOverflow, etc. This application was made to help people voice their concerns about the community! It shows pins dropped by users that have decided that there is a problem that needs to be addressed by the city, and the more the users drop pins in the same area, the redder it gets; making it a severe problem that has to be addressed asap. There are 2 pages, the map and the new post screen! Perfect for phone broswers. Works perfectly for android and iPhone! We also made the data stored in our database into an API that can be accessed by anyone!
